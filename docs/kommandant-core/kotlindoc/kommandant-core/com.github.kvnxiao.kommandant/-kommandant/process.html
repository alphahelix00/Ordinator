process

open fun <T> process(input: String, vararg opt: Any?): CommandResult<T> (source)

Processes a string input with any additional variables for command execution.

Parameters

input - The string input to parse into valid context and command.

opt - A nullable vararg of optional, extra input variables.

Return
CommandResult The result after attempting execution of the command. Will return false and null if there was no command to execute.
