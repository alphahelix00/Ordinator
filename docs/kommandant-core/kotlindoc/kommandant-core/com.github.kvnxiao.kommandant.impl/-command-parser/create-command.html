Overrides ICommandParser.createCommand

Creates a command by parsing a single CommandAnn annotation, with its execution method set as the method targeted by the annotation.

Parameters

instance - The instance object for which its class is to be parsed.

method - The method to invoke for command execution.

annotation - The annotation to parse.

Return
