executeCommand

protected open fun <T> executeCommand(command: ICommand<*>, context: CommandContext, vararg opt: Any?): T (source)

Runs the command's execution method defined by its ICommandExecutable.

Because the command in the parameter can have a method return of any generic type, we require an unchecked cast to convert from a wildcard to specific type T. We operate under assumption that the user knows what they are doing and that the command should have a specific return type of type T.

Parameters

command - The command to execute.

context - The context of the command, containing the calling alias and any args it may have.

opt - A nullable vararg of Any, which can be useful in specific implementations when a command requires more than just context for execution.

Exceptions

InvocationTargetException - When the method invocation failed.

IllegalAccessException - When the method could not be accessed through reflection.

Return
T Returns the result of the execution.
