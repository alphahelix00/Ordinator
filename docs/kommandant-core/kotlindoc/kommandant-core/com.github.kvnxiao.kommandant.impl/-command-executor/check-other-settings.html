checkOtherSettings

protected open fun checkOtherSettings(command: ICommand<*>, context: CommandContext, vararg opt: Any?): Boolean (source)

An extra method that can be used in other implementations to check whether it is valid to execute the command. Akin to a "permission" check.

Parameters

command - The command to execute.

context - The context of the command, containing the calling alias and any args it may have.

opt - A nullable vararg of Any, which can be useful in specific implementations when a command requires more than just context for execution.

Return
Boolean Defaults to true in the core library. This method can be overrided in other project implementations.
