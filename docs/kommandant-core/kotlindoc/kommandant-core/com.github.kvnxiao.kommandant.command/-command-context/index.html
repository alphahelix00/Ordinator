CommandContext

open class CommandContext (source)

A data class for containing the context of the command. It parses a string and splits it in two based on the first available space. The first string represents the alias, and the second string represents the arguments leftover.

NOTE: This class was written in Java. And why, you might ask? As of Kotlin 1.10, there is still some performance overhead in splitting strings by a literal. As such, string splitting in Java is currently still more performant than in Kotlin.

<init> CommandContext(message: String?)

Constructor which takes in a string to split in two. Splits based on the first available space character. If a space does not exist, the args string will be null. If the input string is null, both resultant args and aliases will be null.
