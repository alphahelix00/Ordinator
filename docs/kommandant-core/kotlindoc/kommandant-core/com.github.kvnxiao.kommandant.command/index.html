CommandBuilder open class CommandBuilder<T>

A builder class that helps define a command without needing to specify all the properties for the ICommand constructor.
CommandContext data class CommandContext

A data class for holding the context of the command. It parses a string and splits it in two based on the first available space. The first string represents the alias, and the second string represents the arguments leftover.
CommandDefaults object CommandDefaults

An object class containing the default command property settings.

A data class containing the properties of the command.

ICommand abstract class ICommand<T> : ICommandExecutable<T>

An abstract class which represents a command. All commands implement ICommandExecutable as their main executing method or function.
ICommandExecutable interface ICommandExecutable<out T>

A functional interface that represents the executing method or function for a command.

Annotations

CommandAnn annotation class CommandAnn

An annotation class that should be used by classes implementing ICommandParser (e.g. CommandParser), which will process classes for methods annotated with this annotation and create new commands from them.
