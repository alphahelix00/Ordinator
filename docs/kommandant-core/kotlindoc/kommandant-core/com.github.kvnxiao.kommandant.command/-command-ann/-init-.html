CommandAnn(prefix: String = CommandDefaults.PREFIX, uniqueName: String, aliases: Array<String>, description: String = CommandDefaults.NO_DESCRIPTION, usage: String = CommandDefaults.NO_USAGE, parentName: String = CommandDefaults.PARENT, execWithSubcommands: Boolean = CommandDefaults.EXEC_WITH_SUBCOMMANDS, isDisabled: Boolean = CommandDefaults.IS_DISABLED)

An annotation class that should be used by classes implementing ICommandParser (e.g. CommandParser), which will process classes for methods annotated with this annotation and create new commands from them.

The annotation should target methods with the same signature as ICommandExecutable to define commands and their properties.
