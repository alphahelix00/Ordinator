CommandAnn

@Target([AnnotationTarget.FUNCTION]) annotation class CommandAnn (source)

An annotation class that should be used by classes implementing ICommandParser (e.g. CommandParser), which will process classes for methods annotated with this annotation and create new commands from them.

The annotation should target methods with the same signature as ICommandExecutable to define commands and their properties.

Defines whether the command should execute alongside its subcommand or be skipped when subcommands are processed. Defaults to CommandDefaults.EXEC_WITH_SUBCOMMANDS.

Defines whether the command is disabled. Defaults to CommandDefaults.IS_DISABLED.

Defines the unique name of the parent command which implies that this command will be a subcommand of that parent. This is used to link subcommands to their parent commands. Defaults to CommandDefaults.PARENT as an empty string representing no parent command.
