processAsync

open fun <T> processAsync(input: String, vararg opt: Any?): Deferred<CommandResult<T>> (source)

Processes a string input with any additional variables for command execution asynchronously. This wraps the original process function with a coroutine which returns a deferred command result.

Parameters

input - The string input to parse into valid context and command.

opt - A nullable vararg of optional, extra input variables.

Return
Deferred The deferred result after attempting execution of the command asynchronously.
