addCommand

open fun addCommand(command: ICommand<*>): Boolean (source)

Overrided addCommand method which adds a command to the registry and additionally checks for enableConfigReadWrite value in attempt to load command properties from JSON if true. If the JSON does not exist, it will be created with the command's default properties.

Parameters

command - The command to add to the registry.

Return
Boolean Whether adding the command was successful or not.
